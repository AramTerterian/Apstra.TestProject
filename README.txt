PURPOSE:
The progam calculates a most optimal subnet mask (minimal number of nodes) 
for a list of IP adresses.

----------------------------------------------------------------------------

INPUT DATA:
Fill in the "input.txt" file with a list of IP adresses (1 or more items) in 
the next format:
X.X.X.X
X.X.X.X
...

For example:
192.168.1.1
192.168.1.98
...

-----------------------------------------------------------------------------

START A PROGRAM:
Execute the "Apstra.TestProject" program file.

-----------------------------------------------------------------------------

OUTPUT DATA:
The result is a subnet mask in the file "output.txt" in the format:
X.X.X.X

For example:
255.255.255.128
